第一节：java最基础回顾（资料：李老师视频和文档、PPT）
	1、了解java平台（什么是java？java平台包括哪些内容？java能做什么？）
		① java是 Sun Microsystems公司与95年推出的Java程序设计语言和Java平台的总称(96年推出第一个正式版本)。
			首先，Java是一门编程语言，Java能够创建所有其它传统语言能填写的应用程序，运行在装有Java虚拟机的操作系统上；
			其次，Java还是一个开发环境，Java技术包含一系列的工具：编译器javac 文档生成工具jdoc，打包工具 jar等，还包含一个完整的Java集合(基本语言类、GUI控件等)；
			最后，Java也是一个运行环境，由Java JDK提供的JRE（Java Runtime Envioument）来搭建运行环境。
		② java是一个完整的平台，不仅提供有些的编程语言，还提供大量的可重用代码及一个安全性、可移植性、自动垃圾回收等系统服务的执行环境。
		③ java有三个方向：JAVASE-桌面开发和低端商务应用的解决方案；JAVAEE-以企业为环境而开发应用程序的解决方案；JAVAME-致力于消费产品和嵌入式设备的最佳解决方案。
	2、JVM（java虚拟机原理：？--java程序编译和执行的的过程？）
		java程序从源文件创建到程序运行要经过两大步骤：1、源文件有编译器编译成字节码文件；2、字节码由java虚拟机解释运行。
		运行时，将编译后的.class类加载到虚拟机jvm(只有在用的时候才将类加载)通过（字节码校验器-解释器及JIT代码生成器根据不同的硬件来解释执行）。
	3、JAVA 核心机制及特点
		JVM :JAVA虚拟机 是JRE的一部分，是在实际计算机通过仿真模拟出的一个虚构出来的计算机----特点：跨平台运行，使用JVM实心跨平台与操作系统无关。
		GC  :垃圾回收  提供一种系统级线程以跟踪每一存储器的分配情况。在Java虚拟机的空闲周期，垃圾收集线程检查并释放那些可被释放的存储器。
		代码安全性 在运行时，执行机器只是将.class类文件通过交互后使用，一般情况下是看不到它的源代码.Java，加强了代码的安全性。
	4、JDK 安装及环境变量设置
		官网下载稳定版本，配置环境变量。
	5、JAVA 相关知识（包、数据类型、类、变量）
		*  java程序最基本单元  类
		*  java文件中新建的一个类   类名和文件名必须一样 
		*  package  表明类所在包  目录  包名的定义  全是小写字母
		*  在同一源文件中  可以写多个类   在这个多个类中  只能有一个 和源文件名字相同   
		*  和源文件名字相同类 必须前面加上 public 访问控制符；编译后生成多个clss文件。
		*  所有的类都来自object   object 这个是java api 提供一个所有类的根类 所有的类都默认继承了object
		*  数据类型 不同数据类型 在定义变量的时候分配空间不一样大
		*  每一个数据类型的取值范围也不一样大
		*  数据类型之间是可以转换的（自动转换和强制转换）
		*  小的可以直接转换成大的   大的可以强制转换 但是数据会有损失
		*  学习java 语言中 8中基本数据类型
		*  java 语言中数据类型 分类
		*  基本类型
		*  整型（byte short int  long）  浮点型（单精度float 双精度double） 
		*  字符型（char）    boolean
		*  引用类型 类的类型  对象类型 自定义的类型
		*  变量 （是在内存中有个空间 里面的数据是可以改变  但是数据改变范围
		*  取决于变量的类型 java 语言叫做强数据类型语言 在定义变量的时候必须
		*  指明变量的类型）
		*  变量的分类   全局的变量（类的属性） 静态变量  非静态变量  局部变量 （是在某一个区域）  参数变量（局部变量）
		*  对于全局变量来讲  i在编译的时候是会初始化值的 隐士的赋值 
		*  默认值是多少取决于类型  整型默认值：0  浮点型：0.0 char默认 "" boolean 默认false
		*  变量名  都小写 如果有第二个单词 首字大写
		*  在一个类中 变量的名 方法的名字 不能够重复的 不能使用数字开头
		*  在静态的方法中是不能够使用非静态的变量
		*  局部变量的定义
		*  java程序中 变量是必须初始化才能使用
		*  局部的变量  是不会自动初始化 必须程序员自己进行 显示的初始化
